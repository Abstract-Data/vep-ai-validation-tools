Contributing
============

We welcome contributions to UV Template Project!


Development Setup
-----------------


1. Clone the repository:

   .. code-block:: bash

      git clone https://github.com/username/uv-template-project
      cd uv-template-project

2. Install development dependencies:

   .. code-block:: bash

      uv sync --extra dev

3. Run tests:

   .. code-block:: bash

      uv run pytest
